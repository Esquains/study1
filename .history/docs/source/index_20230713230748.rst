PyTorch documentation
===================================

PyTorch is an optimized tensor library for deep learning using GPUs and CPUs.

Features described in this documentation are classified by release status:

  *Stable:*  These functionalities will be consistently supported over the long term, 
  with minimal performance limitations or gaps in documentation. 
  While occasional breaking changes may occur, advance notice will be provided one release ahead of time, 
  and backwards compatibility will generally be maintained.

  *Beta:*  These features are labeled as Beta due to potential API changes based on user feedback, 
  performance enhancements, or incomplete coverage across operators. With Beta features, 
  our commitment is to refine them until they reach Stable status. However, 
  we do not guarantee backwards compatibility for Beta features.

  *Prototype:*  These features are typically not included in binary distributions such as PyPI or Conda, 
  except occasionally accessible through runtime flags. They are in the early stages of development, 
  aiming for feedback and testing purposes
